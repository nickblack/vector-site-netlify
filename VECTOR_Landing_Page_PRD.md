# VECTOR Landing Page - Product Requirements Document (Updated)

## 1. Project Overview

### 1.1 Project Name
VECTOR Landing Page – Deep Tech GTM Conversion Hub

### 1.2 Project Description
A stunning, high-converting landing page for VECTOR, a deep tech founder’s GTM framework. The design will evoke high-end B2B SaaS platforms (e.g., Linear, Stripe, Vercel) with a sophisticated, developer-focused aesthetic. The page will serve as a sub-brand under Nick Black, using consistent navigation and footer, but with a distinct VECTOR visual identity.

### 1.3 Business Objective
Convert qualified deep tech founders into leads through a compelling, modern landing page that positions VECTOR as the go-to commercialization system for technical founders, while reinforcing the connection to the Nick Black brand.

## 2. Design Inspiration & Principles
- Clean, technical layouts with dominant Vector Purple (#BA37F3)
- Modern typography with clear hierarchy
- Interactive elements (subtle fades, reveals, hover effects)
- Professional testimonial sections
- Structured, grid-based framework presentation
- Generous whitespace for clarity and focus

## 3. Features for v1
- Hero section with compelling value proposition
- VECTOR framework breakdown (V-E-C-T-O-R sections)
- Target audience and process explanation
- Success stories with testimonials
- Pricing tiers
- Contact form/primary CTA (lead magnet download)

## 4. Design System
- **Colors:**
  - **Vector Purple (#BA37F3):** Dominant brand color for hero and section backgrounds, gradients, and accents
  - **Primary Dark (#272726):** Header, footer, and navigation backgrounds
  - **Green (#148F7C):** Primary action button (e.g., "Download Playbook")
  - **Turquoise (#1CA6B2):** Secondary action button (e.g., "Book Consultation")
  - **Light Gray (#F0F0F0):** Section backgrounds, cards
  - **White (#FFFFFF):** Backgrounds, whitespace
- **Gradients:** Purple gradients for hero/footer/CTAs
- **Typography:** Inter (fallback for Aeonik), clear hierarchy (bold headlines, readable body)
- **Animations:** Subtle fades and reveals on scroll and hover
- **Layout:** Grid-based, generous whitespace, card-style sections for framework, testimonials, and pricing

## 5. Brand & Visual Integration
- **Header:** Black bar (#272726), Nick Black logo, navigation, and a “VECTOR by Nick Black” sub-brand indicator
- **Footer:** Dark, minimal, matches parent brand, with VECTOR contact if needed
- **Navigation:** Link back to main Nick Black site
- **Sub-branding:** VECTOR visual identity is prominent but clearly sits under Nick Black

## 6. Content & Section Structure
1. **Hero:**
   - Left: Bold headline, subheadline, green CTA (Download Playbook), secondary CTA (Book Consultation)
   - Right: Playbook visual (placeholder or styled box if no asset)
   - Purple gradient background
2. **Problem Statement:**
   - Large, bold headline
   - Supporting text
   - Purple callout band for “VECTOR is built for the complexity of deep tech.”
3. **VECTOR Framework:**
   - Grid of 6 cards (V-E-C-T-O-R), each with icon/letter, title, and description
4. **20+ Proven Models:**
   - Subtitle and 3 testimonial cards (with quote, name, and company)
5. **Deep Tech Segments:**
   - Grid of 6 cards, each with segment name and description
6. **How It Works:**
   - 5-step process, each as a card with icon/title/description
7. **Success Stories:**
   - 3 case study cards with company, quote, and founder name
8. **Pricing:**
   - 3 pricing cards (Intensive, Launch, Coaching) with price and description
9. **Final CTA:**
   - Purple gradient band, headline, subheadline, and green CTA (Download Playbook)
10. **Footer:**
    - Dark, minimal, Nick Black and VECTOR branding, navigation links

## 7. Technical Requirements
- HTML/CSS/JS (no frameworks for v1)
- Responsive and accessible
- Modal for lead magnet form
- Subtle scroll/hover animations
- Analytics hooks for conversion tracking

## 8. Content Strategy
- Authoritative, direct, energetic tone
- Clear, founder-focused messaging
- Consistent with both Nick Black and VECTOR brand voices

## 9. Success Metrics
- Conversion rate for lead magnet downloads
- Engagement (time on page, scroll depth)
- Brand recall/recognition for VECTOR as a Nick Black sub-brand
